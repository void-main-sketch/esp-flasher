✨ Features

Everything You Need

SparkOS packs powerful features into a tiny microcontroller, giving you games, utilities, and system tools.

📱

Interactive Menu

Intuitive navigation system with hardware buttons. Browse through apps, games, and settings with ease.

📶

WiFi Manager

Easy WiFi configuration via captive portal. No hardcoded credentials — connect securely on first boot.

🕐

NTP Time Sync

Automatic time synchronization with configurable timezone. Always displays the correct time.

🎮

Built-in Games

Dodge — avoid falling obstacles. Pong — classic paddle game. Fun on a tiny screen!

🧮

Utilities

Calculator for quick math and a Console showing WiFi status, IP address, and timezone.

💾

Persistent Settings

WiFi credentials and timezone preferences are stored permanently. Survives reboots and power cycles.

🚀 Quick Start

Getting Started

Get SparkOS running in 5 simple steps.

1

Install Libraries

Add required libraries via Arduino Library Manager: U8g2, WiFiManager, NTPClient.

2

Wire Hardware

Connect your 0.96" OLED display (SSD1306, I2C) and 3 push buttons according to the pinout table above.

3

Upload Firmware

Compile and upload to your board using Arduino IDE or use the Web Installer above.

4

First Boot — WiFi Setup

Connect to the ESP-OS WiFi network from your phone/laptop. Configure your WiFi credentials via the captive portal.

5

Enjoy! ⚡

Navigate using UP/DOWN buttons, press EXIT/SELECT to enter apps. Play games, check the time, explore utilities.
